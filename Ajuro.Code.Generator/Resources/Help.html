Ajuro Template Manager

Paste your code in the Original Code panel.
The application helps you define sections of your code that will be used to generate code. There are 2 types of sections so far:

Multiplication Sections

Are used to repeat fragments of code. When you create your selections, if they contain spaces they will be by default Multiplication Sections

Replacement Sections

Are used to substitute the selected text with your variables. Single-word selections are by default Replacement Sections.

How to use it?

Can load sample templates. Create template if not exist.

Complexity Level 0

Paste any fragment of code. Press F5 to inflate the template (same text goes to output).

Complexity Level 1

Replacement/Interpolation fragments

Double click words to make them variables.
You can rename the placeholder.
You can specify the data type to be generated for demo.
Random GUID, Random increasing DateTimes, Random integers from 1 to 10, Random integers from 10 to 100, Random integers from 100 to 1000, Random customors first names, Random customors full names-->

Complexity Level 2

Repetition/Multiplication fragments
Select over another selection.
Rename it if needed. Specify the number of fragments to be generated. For random occurences provide interval.

Complexity Level 3

Skip-Once fragments
Specify a skip-once fragment before a Repetition/Multiplication fragment. Because usualy enumerations does not start/end with their separator.
Select a coma (most freqvent case).
Change it's type to skip-once if needed.

Complexity Level 4

Use custom data to inflate the template.
Replace generated json with your own.
